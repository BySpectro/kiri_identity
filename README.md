# Descarga & Instalacion

# Manual
- Descarga https://github.com/BySpectro/kiri_identity
- Colocalo en tu directorio ESX

# Instalacion
- agrega `esx_identity.sql` a tu base de datos
- agrega esx_identity a tu `server.cfg`:

```
start esx_identity
```

# Comandos
